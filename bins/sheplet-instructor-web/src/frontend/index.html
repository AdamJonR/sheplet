Add Course Materials

Point Sheplet at a folder of your course documents (PDF, Word, Excel, CSV, or plain text). It will read, analyze, and index them so the AI tutor can answer student questions using your materials.

AI Model

Choose and download the AI model that will power your course tutor. Larger models give better answers but need more disk space and memory. All models run entirely on this computer — no internet required after download.

Not sure? Start with Phi-3 Mini. If students have older computers, choose a smaller model.

Customize AI

Teach the AI model to respond in a style and tone that fits your course. You provide example question-answer pairs, and Sheplet adjusts the model to match. This step is optional but can significantly improve answer quality.

Creates starter files with the correct format for your training examples.
"Learn from examples" is simpler to set up. "Learn from comparisons" can fine-tune tone and style if you provide both good and bad answers.
Advanced options
How aggressively the model adjusts. Lower = more cautious. Leave blank for a sensible default.

Tutor Settings

Control how the AI tutor behaves when students ask questions. These settings are locked for students — only you can change them.

Tell the AI how to behave: what tone to use, whether to give direct answers or guide students, and any course-specific rules.
"Best matches" is recommended for most courses. "Diverse matches" works better when students ask broad questions that span multiple topics.
How many passages from your documents the AI reads before answering. More passages = more context but slower responses. 3–7 is typical.

Package & Share

Package everything into a single .sheplet file that you distribute to students. This file contains the AI model, your course materials index, customizations, and settings — everything students need to run the tutor offline.
